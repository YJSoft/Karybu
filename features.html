Features

Opensource

Karybu is free and opensource under the LGPL license. Everyone, from the big web agency to the small internet startup can use Karybu the way they want as a fundamental part of their business. Plus they can fork and improve it.

Visual editing

We want you to have a comfortable editing experience. To achieve this, Karybu offers drag-and-drop functionality so that you can edit your front-end widgets to look just the way you want them to. Just create a widget page and you're all set to start building a beautiful website.

Extensible / Modular structure

Almost every piece of code in our CMS is part of a module, widget or add-on, so we're quite experienced in providing the best features to developers.

Multiple databases support

We support:
  • MySQL (MyISAM and InnoDB), MySQLi, MySQL PDO
  • CUBRID
  • Firebird
  • PostgreSQL
  • SQLite2, SQLite3 PDO
  • MSSQL

Responsive

By using Twitter Bootstrap on both admin and front-end, Karybu is easily accessible from any portable device as well as from a PC.

User friendly dashboard

We built Karybu with the end user in mind, trying to keep things as simple and intuitive as possible.

Fast install

Using our simple installer, you can have your website up and running in under 5 minutes.

Social networking integration

Adapting to social media demands is a must in today's internet context. So you have the possibility to authenticate with Facebook, Twitter and Google Plus. Also our Board module comes with sharing options for your blog, forum or gallery.

Symfony2 components

Symfony allows you to use certain of its software building blocks (dependency injector, http abstraction, event dispatcher) without necessarily using the framework in its entirety. Which is what we did in order to provide developers with the best tools for creating lovely modules.

Twitter Bootstrap

This awesome tool collection included in Karybu contains HTML and CSS-based design templates for typography, forms, buttons, charts, navigation and other interface components, as well as optional JavaScript extensions.

WYSIWYG editors

To have a great experience in content management for your website the editor component has to be simple, intuitive but also feature rich. So we used
CKEditor (default) and TinyMCE, two of the best editors out there.

Debug toolbar and custom environments

Developers will be fond of our debug toolbar as it will be a great companion in developing even greater websites and/or extensions. We also provide two built-in environments for production and development, depending on your website's current status.

Adaptive to your needs

Using our powerful Board module you can adapt your website to your needs and easily create a blog, a forum or just a gallery to share pictures with your visitors.

Use inspiration from other websites

There are a lot of interesting things on the web that serve as inspiration. But when we actually want to publish something, most of us forget not only the URL that was inspiring but also the fact of having found something inspiring in the first place. In this regard you can try our Material (Ingredients Collector) module. All you have to do is drag ingredients / materials such as texts, links, images or videos that you have collected before when publishing something with our editors.
